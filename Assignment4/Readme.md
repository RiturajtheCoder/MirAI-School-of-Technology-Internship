# 🎨 AI Image Studio

AI Image Studio is a Streamlit-based AI image generation application that transforms text prompts into stunning AI-generated artwork using the **Pollinations AI Image API**. The application provides customizable image dimensions, multiple art styles, a Magic Enhance feature, and a fun Surprise Me mode for creative inspiration.

## 🚀 Live Demo

🌐 **Streamlit App:**  
**https://aiimagestudio.streamlit.app**

---

## ✨ Features

- 🎨 Generate AI images from text prompts
- 🖼️ Multiple art styles
- 📏 Adjustable image width and height
- ✨ Magic Enhance mode for higher-quality prompts
- 🎲 Surprise Me feature with random creative prompts
- 📥 Download generated images as PNG
- ⚡ Fast and simple Streamlit interface

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/<your-username>/AI-Image-Studio.git
cd AI-Image-Studio
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
streamlit run app.py
```

---

## 🎮 How to Use

1. Enter a description of the image you want to generate.
2. Select your preferred art style.
3. Adjust the image width and height using the sliders.
4. (Optional) Enable **✨ Magic Enhance** for more detailed artwork.
5. Click **🎨 Generate Image**.
6. Download the generated image as a PNG file.

- ☕ A dragon drinking coffee in a library

## ✨ Magic Enhance

Enable the **Magic Enhance** option to automatically enrich your prompt with professional-quality keywords such as:

- masterpiece
- 8K resolution
- highly detailed
- trending on ArtStation
- Unreal Engine 5 render

This helps generate more visually appealing images.
