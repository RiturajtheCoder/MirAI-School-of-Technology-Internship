# 🌌 AI Multiverse

**AI Multiverse** is a Streamlit-powered AI chatbot that lets users interact with multiple AI personalities using Google's Gemini API. Simply choose a character, enter a message, and receive responses that stay true to the selected personality.

## 🚀 Live Demo

🔗 **https://aimultiverse.streamlit.app/**

---

## ✨ Features

- 🤖 Multiple AI personalities
- 💬 Interactive chat interface
- ⚡ Fast responses powered by Google Gemini 2.5 Flash
- 🎭 Personality-based prompt engineering
- 🌐 Clean and responsive Streamlit UI
- 🔒 Secure API key management using environment variables

---

## 🎭 Available Personalities

- 👨‍💻 An Expert Hacker
- 🏏 Virat Kohli
- 😡 An Angry Ravi Shastri
- ⚽ A Crazy Ronaldo Fan
- 🇺🇸 Donald Trump

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/<your-username>/AI-Multiverse.git
cd AI-Multiverse
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Create a `.env` file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

### Run the application

```bash
streamlit run app.py
```

---

## 🎮 How to Use

1. Launch the application.
2. Select an AI personality.
3. Enter your message.
4. Click **SEND**.
5. Receive a response generated by Gemini in the chosen personality.
